# Orion Platform

Orion is a distributed event, task, and notification management platform
designed with microservice architecture principles.

## Goals
- Practice real-world microservice patterns
- Learn event-driven architecture
- Apply Elasticsearch, caching, messaging, and observability
- Build a production-like system end-to-end

## Planned Architecture
- API Gateway
- Auth Service
- User Service
- Event Service
- Task Service
- Notification Service
- Search Service

## Technology Stack (Planned)
- Spring Boot / Spring Cloud
- PostgreSQL
- RabbitMQ
- Elasticsearch
- Redis
- Docker & Docker Compose
- Keycloak
- Prometheus & Grafana
